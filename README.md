# ByteBites — Frontend Learning Project (HTML, CSS, TailwindCSS)

ByteBites is a beginner-friendly, real-world styled restaurant website project used to teach:
- Semantic HTML + accessibility basics
- CSS fundamentals (box model, typography, flexbox, grid, responsive design)
- TailwindCSS utility-first workflow (responsive utilities, hover/focus states, layout)

Students build the site in **two versions**:
1) **Vanilla** (HTML + CSS)
2) **Tailwind** (HTML + Tailwind)

---

## Project Pages
### 1) Home (`index.html`)
Required sections:
- Sticky header/nav (logo, links, CTA)
- Hero (headline, text, 2 CTAs, image/placeholder)
- Featured dishes (card grid)
- Testimonials (2–3)
- Footer (hours/location + copyright)

### 2) Menu (`menu.html`)
Required sections:
- Category tabs/filters (UI only, no JS required)
- Specials banner
- Menu grid/list (name, description, price)

### 3) Contact/Reservations (`contact.html`)
Required sections:
- Contact info panel
- Map placeholder block (no external integration required)
- Reservation form (name, email, date, time, guests, notes)
